# Mellisa Goodwin - Official Website

Professional actress portfolio website for Mellisa Goodwin.

## About

Mellisa Goodwin is a professional actress based in Los Angeles, California. She is known for her work in:

- **Law & Order** (TV Series)
- **Law & Order: Special Victims Unit** - Counselor Janet Mayer
- **We Never Sleep** (2023) - Mikaela
- **Wetlands** (2017) - Lisa

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Deployment

Deploy easily on Vercel:

```bash
npm run build
```
